How to Use This Tool

Find & Replace

Type into the "Find" box to highlight matching text. The editor will get a red outline to show this feature is active.

  • Enter: Jump to the next match.
  • Shift + Enter: Jump to the previous match.
  • Ctrl/Cmd + Enter: Replace the current match and jump to the next one.
  • Click the Red (X) button to clear the inputs and exit Find mode.

Links

Highlight text and click the Link () button to add a URL. If you click on an existing link, the button will turn blue, allowing you to update or remove the link.

AI Artifacts

Use the Yellow Square () button to highlight invisible AI characters. This adds a yellow outline to the editor. Use the Trash button to remove them. (If both Find and AI are active, the outline is orange).

Markdown Mode

Toggle Markdown mode to see your text as code. This tool supports **bold**, *italic*, __underline__, and ~~strikethrough~~. Text alignment (center/right) is also preserved when toggling.

Tips & Known Issues

  • Typing Lag: If your typing feels stuck, make sure the "Find" box is empty and the AI Highlight is off (check for the colored outline).
  • Downloading: You can save your work as a .html file (which preserves all formatting) or a .txt file (plain text).
