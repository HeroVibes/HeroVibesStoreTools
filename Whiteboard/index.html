How to Use Scratchpad

    This is your personal whiteboard. Add notes, links, code snippets, and images and arrange them however you like.

    • Add Note: Click 'Add Note' or press N to create a blank note card.
    • Paste Content: Paste a link, code, or image URL into the input bar and press Enter. You can also paste directly onto the canvas (Ctrl+V).
    • Upload: Click 'Upload' or drag-and-drop an image file onto the canvas.
    • Move: Click and drag any card's header to move it. Hold Ctrl while dragging to snap to the grid.
    • Pages: Use the controls on the right to add new pages or switch between them. Double-click a page name to rename it.
    • Code Editing: Double-click a code block to enter edit mode. Click outside the block to save.
    • Download/Copy: Use the buttons on each card to download or copy its content.
