How to Use Your Scratchpad

    Add Content:

    • Click Add Note or press N to create a text note.
    • Paste any link, code, or text — it auto-detects type!
    • Drag & drop images/audio or use the Upload button.

    Organize:

    • Drag cards by the top bar to move them (or tap & hold on mobile).
    • Resize cards by dragging the bottom-left corner.
    • Double-click Code cards to edit inline.
    • Rename cards by clicking their label (Note, Code, etc.).

    Pages:

    • Add new pages with the + button.
    • Double-click page name to rename.
    • Hover active page dot → turns red → click to delete (with confirmation).

    Save & Share:

    • Export saves your entire whiteboard as a .json file.
    • Import loads it back — perfect for sharing with friends!

    Per-Card Actions:

    • Download content
    • Copy to clipboard
    • Delete card
